CarbonXS GUI

Introduction

CarbonXS GUI is intended to be a GUI wrapper around the CarbonXS program for fitting XRD patterns of carbon materials.

Current Version:

Features include:

  • Cross platform support for Windows (Windows 7+), Mac OSX (10.10+), and Linux
  • Export of configuration files into a format that CarbonXS can read.
  • Decoupling of fit parameters, fitting settings, and diffractometer settings into separate files which can be import and exported through JSON.
  • Instant visualization of fit result, background, and fit difference.
  • Graceful handling of CarbonXS's errors and limitations.
